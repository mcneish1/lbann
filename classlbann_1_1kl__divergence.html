Detailed Description

Kullback Leibler (KL) divergence regularizer It compute latent loss between the distribution of latent space induced by an encoder on input data and some prior. It is acts as a kind of regularizer added to reconstruction cost. A good use is in Variational Autoencoder (VAE). kl_loss = -0.5 * mean(1 + z_log_sigma - square(z_mean) - exp(z_log_sigma)) See the following reference: Kingma and Welling, "Auto-Encoding Variational Bayes" Doersch, "Tutorial on Variational Autoencoders" https://blogs.keras.io/building-autoencoder-in-keras.html TODO: Add checkpoint
